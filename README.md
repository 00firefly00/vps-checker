# vps-checker
VPS Service Checker
VPS Service Checker — это простой скрипт для Ubuntu VPS, который позволяет:
Проверять доступность популярных стриминговых сервисов: Netflix, YouTube, Disney+, TikTok, Spotify и ChatGPT.
Отображать статус сервисов с галочками ✔ / крестиками ✖ и определять регион доступа.
Определять геолокацию IP через несколько источников и подсвечивать расхождения.
Проверять IP на попадание в blacklist (Spamhaus, SORBS, AbuseIPDB).
Проводить тест скорости интернета отдельным пунктом меню (Speedtest).
Красиво выводить результаты в цветной и компактной таблице, удобно для смартфона и терминала.
