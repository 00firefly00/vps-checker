
# VPS Checker

Простой и удобный Bash-скрипт для проверки VPS: стриминг-сервисов, геолокации IP и blacklist.

## 🚀 Возможности

- 📺 Проверка доступности сервисов:
  - Netflix
  - YouTube
  - Disney+
  - TikTok
  - Spotify
  - ChatGPT

- 🌍 Определение региона для каждого сервиса

- 🧭 GeoIP проверка через несколько баз:
  - ipinfo
  - ip-api
  - ipwhois
  - ipapi
  - ifconfig

- ⚠️ Обнаружение расхождений GeoIP (важно для Netflix и антибот-систем)

- 🛡 Проверка IP по blacklist:
  - Spamhaus
  - SORBS
  - AbuseIPDB

- ⚡ Тест скорости интернета (отдельный пункт меню)

- 🎨 Цветной и компактный вывод (удобно даже с телефона)
